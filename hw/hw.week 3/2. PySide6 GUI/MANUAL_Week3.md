## 3. Lab별 사용법

### Lab 1: Perceptron

**뭘 하는 건지:** 뇌세포 1개로 AND/OR/XOR 논리 게이트를 학습시켜 봅니다.

**조절할 수 있는 것:**
- 학습률 (0.01~1.0): 한 번에 얼마나 크게 고칠지. 크면 빠르지만 불안정
- Epochs (10~5000): 학습 반복 횟수. 많을수록 정확하지만 느림

**사용 순서:**
1. 학습률과 Epochs를 원하는 값으로 설정
2. 초록색 `학습 실행` 버튼 클릭
3. 왼쪽 그래프 3개 확인 (AND, OR, XOR)
4. 오른쪽 로그에서 각 게이트의 예측/정답/오류 수 확인

**그래프 읽는 법:**
- 파란색 영역 = "0이다" 판단 구역
- 빨간색 영역 = "1이다" 판단 구역
- X 표시 = 정답이 0인 점
- 빨간 동그라미 = 정답이 1인 점
- AND/OR: 깔끔하게 나뉨 (성공!)
- XOR: 엉망 (실패! → Lab 4에서 해결)

**해볼 것:** 학습률을 0.01로 낮추고 Epochs를 10으로 줄여보세요. AND도 학습이 안 될 수 있어요!

### Lab 2: Activation Functions

**뭘 하는 건지:** 4가지 활성화 함수의 모양과 미분(기울기)을 비교합니다.

**조절할 수 있는 것:**
- 체크박스 4개: Sigmoid, Tanh, ReLU, Leaky ReLU (켜고 끄기)
- Leaky α 값 (0.001~0.5): Leaky ReLU에서 음수 기울기

**사용 순서:**
1. 체크박스로 보고 싶은 함수를 선택
2. 그래프가 자동으로 갱신됨 (버튼 안 눌러도 됨!)
3. 왼쪽 = 함수 모양, 오른쪽 = 미분(기울기) 비교

**그래프 읽는 법:**
- 왼쪽 그래프: x축 = 입력값, y축 = 출력값
- 오른쪽 그래프: x축 = 입력값, y축 = 기울기 크기
  - 기울기가 0이면 학습이 멈춤!
  - Sigmoid/Tanh는 양쪽 끝에서 기울기 ≈ 0 (문제!)
  - ReLU는 오른쪽에서 기울기 항상 1 (좋음!)

**해볼 것:**
- Sigmoid만 켜고 나머지 끄면 S자 곡선이 확실히 보여요
- Leaky α를 0.3으로 올려보면 음수 영역 기울기가 확 커져요

### Lab 3: Forward Propagation

**뭘 하는 건지:** 숫자를 신경망에 넣으면 안에서 어떤 계산이 일어나는지 한 단계씩 보여줍니다.

**조절할 수 있는 것:**
- x1 (-2.0~2.0): 첫 번째 입력값
- x2 (-2.0~2.0): 두 번째 입력값

**사용 순서:**
1. x1, x2 값을 원하는 숫자로 설정
2. 파란색 `Forward Pass 실행` 버튼 클릭
3. 왼쪽: 네트워크 다이어그램 (동그라미 안에 각 뉴런의 값)
4. 가운데: 각 값의 크기를 막대 그래프로 표시
5. 오른쪽 로그: z1 = X @ W1 + b1 계산 과정 출력

**그래프 읽는 법:**
- 왼쪽 네트워크 다이어그램:
  - 파란 동그라미 = 입력 (x1, x2)
  - 초록 동그라미 = 은닉 뉴런 (ReLU 통과 후 값)
  - 분홍 동그라미 = 값이 0인 뉴런 (ReLU가 음수를 잘라서 죽은 것!)
  - 노란 동그라미 = 최종 출력 (Sigmoid 통과)
  - 선 두께 = 가중치의 크기 (굵을수록 중요한 연결)
- 오른쪽 막대: z1(주황) → a1(초록) → z2(주황) → a2(빨강) 순서

**해볼 것:**
- x1=0, x2=0으로 바꿔보세요. 뉴런 값이 많이 달라져요
- x1=1.5, x2=-1.0으로 바꿔보세요. 다른 뉴런이 죽을 수도 있어요

### Lab 4: MLP (XOR)

**뭘 하는 건지:** Lab 1에서 못 풀었던 XOR을 여러 층 신경망(MLP)으로 해결합니다.

**조절할 수 있는 것:**
- 은닉 뉴런 (2~32): 중간 층 뉴런 수. 많을수록 복잡한 패턴 학습 가능
- 학습률 (0.01~5.0): 한 번에 고치는 크기
- Epochs (100~50000): 학습 반복 횟수

**사용 순서:**
1. 은닉 뉴런, 학습률, Epochs 설정
2. 보라색 `학습 시작` 버튼 클릭 (몇 초 걸릴 수 있음)
3. 왼쪽: Loss(오차) 감소 그래프
4. 가운데: XOR 결정 경계 (곡선이면 성공!)
5. 오른쪽: 은닉층 뉴런별 반응 히트맵
6. 로그에서 정확도 100% 확인

**그래프 읽는 법:**
- Training Loss: y축이 Loss(오차). 아래로 내려갈수록 잘 학습된 것
- XOR Decision Boundary: 빨간 영역(1 판단), 파란 영역(0 판단)
  - Lab 1과 비교: 직선 → 곡선으로 바뀐 것이 핵심!
  - 빨간 동그라미가 빨간 영역에, X가 파란 영역에 있으면 성공
- Hidden Activations: 밝은색(노란) = 강하게 반응, 어두운색(보라) = 약하게 반응

**해볼 것:**
- 은닉 뉴런을 2로 줄여보세요. 가끔 학습이 실패해요
- 학습률을 3.0으로 올려보세요. 학습이 불안정해질 수 있어요
- Epochs를 500으로 줄여보세요. 아직 학습이 덜 돼서 정확도가 낮아요

### Lab 5: Universal Approximation

**뭘 하는 건지:** 뉴런 수를 바꿔가며 함수를 얼마나 잘 흉내내는지 비교합니다.

**조절할 수 있는 것:**
- 목표 함수: sin(2πx), Step Function, Complex (드롭다운 선택)
- 뉴런 수 (1~100): 은닉층 뉴런 개수
- Epochs (500~20000): 학습 반복 횟수

**버튼 2개:**
- `근사 실행` (주황): 현재 설정으로 단일 근사 실행
- `3 vs 10 vs 50 비교` (회색): 뉴런 3/10/50개를 한 번에 비교

**사용 순서:**
1. 목표 함수 선택
2. `3 vs 10 vs 50 비교` 버튼 클릭 (가장 재밌음!)
3. 3개 그래프가 나란히 나옴
4. MSE(오차) 숫자가 작을수록 잘 맞춘 것

**그래프 읽는 법:**
- 파란 실선 (True) = 진짜 함수 (이걸 맞추는 게 목표)
- 빨간 점선 (NN) = 신경망이 흉내낸 결과
- 초록 점 = 학습에 사용한 데이터
- MSE = 오차. 작을수록 좋음
- 두 선이 겹칠수록 잘 근사한 것!

**해볼 것:**
- 뉴런 수를 1로 설정하고 실행해보세요. 직선 하나밖에 못 그어요!
- 뉴런 50개로 하고 Epochs를 15000으로 늘려보세요. 잘 맞출 수 있어요
- 목표 함수를 바꿔가며 어떤 함수가 더 어려운지 비교해보세요
