## 3. 각 기능 상세

### F-01: Lab 1 — Perceptron

**입력 컨트롤:**
- 학습률 (0.01 ~ 1.0, 기본값 0.1)
- Epochs (10 ~ 5000, 기본값 100)

**출력:**
- AND/OR/XOR 3개 결정 경계 그래프 (파란색=0, 빨간색=1 영역)
- 각 게이트별 예측/정답/오류 수 로그

**핵심 학습 포인트:**
- AND/OR → 오류 0/4 (직선 분리 가능)
- XOR → 오류 2~3/4 (직선 분리 불가능 → Multi-Layer 필요)

### F-02: Lab 2 — Activation Functions

**입력 컨트롤:**
- Sigmoid / Tanh / ReLU / Leaky ReLU 체크박스 (개별 on/off)
- Leaky ReLU의 α 값 (0.001 ~ 0.5, 기본값 0.01)

**출력:**
- 왼쪽: 활성화 함수 그래프 비교
- 오른쪽: 미분(Gradient) 그래프 비교

**핵심 학습 포인트:**
- 활성화 함수가 없으면 여러 층 쌓아도 선형 → 비선형 변환 필수
- Sigmoid: 0~1 출력 → 확률 판단용
- ReLU: 빠르고 기울기 안 죽음 → 중간 층 기본값
- Leaky ReLU: Dying ReLU 문제 해결

### F-03: Lab 3 — Forward Propagation

**입력 컨트롤:**
- x1 (-2.0 ~ 2.0, 기본값 0.5)
- x2 (-2.0 ~ 2.0, 기본값 0.8)

**출력:**
- 네트워크 다이어그램 (뉴런별 값 표시, 가중치 두께로 연결선 표현)
- Layer Values 막대 그래프 (z1, a1, z2, a2)
- 계산 과정 로그 (z1 = X @ W1 + b1 단계별)

**핵심 학습 포인트:**
- z1 = X @ W1 + b1 각 기호의 의미 (X=입력, W1=가중치, @=행렬곱, b1=편향)
- ReLU가 음수를 0으로 자르는 것을 실제 값으로 확인
- 가중치는 랜덤 시작 → Backprop으로 조정

### F-04: Lab 4 — MLP (XOR)

**입력 컨트롤:**
- 은닉 뉴런 수 (2 ~ 32, 기본값 4)
- 학습률 (0.01 ~ 5.0, 기본값 0.5)
- Epochs (100 ~ 50000, 기본값 10000)

**출력:**
- Training Loss 그래프 (로그 스케일)
- XOR Decision Boundary (곡선 결정 경계)
- Hidden Activations 히트맵 (뉴런별 반응 패턴)
- 학습 결과 로그 (예측값, 정확도)

**핵심 학습 포인트:**
- Lab 1의 직선 → Lab 4의 곡선으로 결정 경계 변화
- Backpropagation 4단계: Forward → Loss → Backward → Update
- 정확도 100% 달성 (XOR 해결!)

### F-05: Lab 5 — Universal Approximation

**입력 컨트롤:**
- 목표 함수 선택 (sin, Step, Complex)
- 뉴런 수 (1 ~ 100, 기본값 10)
- Epochs (500 ~ 20000, 기본값 5000)

**출력:**
- 단일 근사 그래프 (True vs NN 비교)
- "3 vs 10 vs 50 비교" 모드 (3개 그래프 동시 출력)
- MSE 값 표시

**핵심 학습 포인트:**
- 뉴런 수 증가 → 근사 정확도 향상 (일반적으로)
- 50개 뉴런이 오히려 못할 수 있음 (학습률/Epochs 조절 필요)
- Cybenko (1989): 충분한 뉴런 → 어떤 연속 함수도 근사 가능
