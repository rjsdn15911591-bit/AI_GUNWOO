# PRD (Product Requirements Document)
## Week 5 딥러닝 핵심 개념 실습 앱

**버전:** 1.0  
**작성일:** 2026-04-15  
**플랫폼:** Windows / macOS / Linux (PySide6)

---

## 1. 제품 개요

### 1.1 목적

Week 5에서 학습한 5가지 딥러닝 핵심 개념(정규화, 과적합/과소적합, 데이터 증강, 전이 학습, CNN)을 **인터랙티브한 GUI 환경**에서 실습할 수 있는 교육용 데스크탑 애플리케이션.

### 1.2 배경 및 동기

- 딥러닝 개념은 수식과 코드만으로 이해하기 어려움
- 파라미터 변화에 따른 결과를 **실시간**으로 시각화하면 직관적 이해 가능
- Python 생태계(TensorFlow, matplotlib)를 PySide6 GUI와 통합하여 학습 효과 극대화

### 1.3 개발 방법론

Claude Code Superpowers 스킬(brainstorming → writing-plans → executing-plans)을 통해 설계-계획-구현 워크플로우를 적용하였다.

---

## 2. 사용자 및 목표

### 2.1 대상 사용자

| 사용자 유형 | 설명 |
|-----------|------|
| 딥러닝 입문자 | Week 5 개념을 처음 학습하는 학부생 |
| 실습 보조 도구 사용자 | 수업 스크립트를 GUI로 실험하고 싶은 학생 |

### 2.2 핵심 사용 시나리오

1. Lab 1: 체크박스로 정규화 기법을 선택하고 Val Loss 곡선을 비교한다
2. Lab 3: 슬라이더를 조작하며 증강 강도에 따른 이미지 변화를 즉시 확인한다
3. Lab 5: CNN 학습 후 예측 그리드에서 오분류 사례를 육안으로 확인한다
