## Lab 1 - 1D 함수 근사

![결과](1-1.%20output/perfect_1d_approximation.png)

### 실험 목적

신경망이 수학 함수를 근사할 수 있는지 확인한다. 네트워크 크기에 따른 근사 성능 차이를 비교한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 함수 | sin(x) |
| 네트워크 | Large [128, 128] |
| 활성화 함수 | tanh |
| Epochs | 3000 |
| 최종 MSE | 0.000013 |

### 결과 및 분석

- 왼쪽 그래프: NN 예측(빨간 점선)이 실제 sin(x) 함수(파란 실선)와 거의 완벽하게 일치한다.
- 오른쪽 그래프: Train Loss와 Val Loss가 함께 수렴하며 10⁻⁵ 수준까지 낮아졌다.
- 최종 MSE가 0.000013으로 매우 낮아, Large [128, 128] 네트워크가 sin(x)를 충분히 근사할 수 있음을 확인했다.
- 네트워크 크기가 클수록 복잡한 함수를 더 정밀하게 표현할 수 있지만, 그만큼 학습 시간도 증가한다.

## Lab 2 - 포물선 운동 회귀

![결과](1-1.%20output/02_projectile_trajectories.png)

### 실험 목적

물리 법칙(포물선 운동)을 신경망으로 학습하여 초기 조건(v₀, θ, t)만으로 위치(x, y)를 예측할 수 있는지 확인한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 입력 | v₀, θ, t |
| 출력 | x, y |
| 네트워크 | [128, 64, 32, 2] + Dropout |
| 학습 샘플 | 2000개 |
| Epochs | 1000 |

### 결과 및 분석

- 왼쪽 그래프: 3가지 조건(v₀=20/30/40 m/s, θ=30°/45°/60°)에 대해 실제 궤적(실선)과 예측 궤적(점선)을 비교했다.
- 예측 궤적이 실제 궤적의 전체적인 형태를 잘 따라가나, 일부 구간에서 편차가 발생한다.
- 오른쪽 그래프: Train Loss는 잘 수렴했으나 Val Loss가 상대적으로 높아, 학습 데이터 이외의 조건에서는 오차가 다소 증가함을 알 수 있다.
- 데이터 수를 늘리거나 Epochs를 증가시키면 예측 정확도가 향상될 것으로 예상된다.

## Lab 3 - 과적합/과소적합 비교

![결과](1-1.%20output/03_overfitting_comparison.png)

### 실험 목적

동일한 데이터에 대해 모델 크기를 달리하여 과소적합(Underfit), 적절한 적합(Good Fit), 과적합(Overfit) 현상을 직접 비교 관찰한다.

### 실험 설정

| 모델 | 구성 | Train Loss | Val Loss |
|------|------|-----------|---------|
| Underfit | [4] | 0.3409 | 0.2789 |
| Good Fit | [32, 16] + Dropout(0.2) | 0.1111 | 0.1181 |
| Overfit | [256, 128, 64, 32] | 0.0748 | 0.0786 |

### 결과 및 분석

- **Underfit [4]**: 뉴런 수가 너무 적어 표현력이 부족하다. 함수의 전체적인 추세는 따라가지만 진동 패턴을 제대로 학습하지 못한다. Train/Val Loss 모두 높다.
- **Good Fit [32, 16] + Dropout**: 실제 함수와 잘 일치하며, Train Loss(0.1111)와 Val Loss(0.1181)의 차이가 작아 일반화 성능이 우수하다.
- **Overfit [256, 128, 64, 32]**: 가장 낮은 Train Loss(0.0748)를 보이나, 200 Epochs 내에서는 Val Loss도 비슷하게 낮아 심각한 과적합은 아직 발생하지 않았다. Epochs를 더 늘리면 Val Loss가 다시 증가하는 전형적인 과적합 패턴이 나타날 것으로 예상된다.
- Dropout 적용 여부가 일반화 성능에 직접적인 영향을 미침을 확인했다.

## Lab 4 - 진자 주기 예측 + RK4 시뮬레이션

![결과](1-1.%20output/04_pendulum_prediction.png)

### 실험 목적

진자의 길이(L)와 초기 각도(θ₀)를 입력받아 주기 T를 예측하고, RK4 수치적분으로 진자 운동을 시뮬레이션한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 입력 | L (0.3~3.0m), θ₀ (5°~80°) |
| 출력 | 주기 T (s) |
| 물리 공식 | T = 2π√(L/g) + 큰 각도 타원적분 보정 |
| RK4 조건 | L=1m, θ₀=30°, dt=0.001s |

### 결과 및 분석

- **좌상단 (주기 예측)**: L=0.5m, 1.0m, 2.0m 각각에 대해 NN 예측(점)이 실제 값(실선)과 매우 잘 일치한다. 각도가 증가할수록 주기가 길어지는 비선형 관계도 잘 포착했다.
- **우상단 (학습 곡선)**: Train Loss와 Val Loss 모두 안정적으로 수렴한다.
- **좌하단 (RK4 시뮬레이션)**: L=1m, θ₀=30° 조건에서 각도가 시간에 따라 규칙적인 진동을 보인다. 감쇠 없는 이상적인 진자 운동이 정확히 재현됐다.
- **우하단 (위상 공간)**: 각도(θ)와 각속도(ω)의 관계가 닫힌 타원형 궤적을 형성한다. 에너지가 보존되는 이상적인 진자의 위상 공간 특성을 잘 나타낸다.
