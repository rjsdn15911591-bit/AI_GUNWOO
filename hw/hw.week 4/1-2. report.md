물리 법칙(포물선 운동)을 신경망으로 학습하여 초기 조건(v₀, θ, t)만으로 위치(x, y)를 예측할 수 있는지 확인한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 입력 | v₀, θ, t |
| 출력 | x, y |
| 네트워크 | [128, 64, 32, 2] + Dropout |
| 학습 샘플 | 2000개 |
| Epochs | 1000 |

### 결과 및 분석

- 왼쪽 그래프: 3가지 조건(v₀=20/30/40 m/s, θ=30°/45°/60°)에 대해 실제 궤적(실선)과 예측 궤적(점선)을 비교했다.
- 예측 궤적이 실제 궤적의 전체적인 형태를 잘 따라가나, 일부 구간에서 편차가 발생한다.
- 오른쪽 그래프: Train Loss는 잘 수렴했으나 Val Loss가 상대적으로 높아, 학습 데이터 이외의 조건에서는 오차가 다소 증가함을 알 수 있다.
- 데이터 수를 늘리거나 Epochs를 증가시키면 예측 정확도가 향상될 것으로 예상된다.

## Lab 3 - 과적합/과소적합 비교

![결과](1-1.%20output/03_overfitting_comparison.png)

### 실험 목적

동일한 데이터에 대해 모델 크기를 달리하여 과소적합(Underfit), 적절한 적합(Good Fit), 과적합(Overfit) 현상을 직접 비교 관찰한다.

### 실험 설정

| 모델 | 구성 | Train Loss | Val Loss |
|------|------|-----------|---------|
| Underfit | [4] | 0.3409 | 0.2789 |
| Good Fit | [32, 16] + Dropout(0.2) | 0.1111 | 0.1181 |
| Overfit | [256, 128, 64, 32] | 0.0748 | 0.0786 |

### 결과 및 분석

- **Underfit [4]**: 뉴런 수가 너무 적어 표현력이 부족하다. 함수의 전체적인 추세는 따라가지만 진동 패턴을 제대로 학습하지 못한다. Train/Val Loss 모두 높다.
- **Good Fit [32, 16] + Dropout**: 실제 함수와 잘 일치하며, Train Loss(0.1111)와 Val Loss(0.1181)의 차이가 작아 일반화 성능이 우수하다.
- **Overfit [256, 128, 64, 32]**: 가장 낮은 Train Loss(0.0748)를 보이나, 200 Epochs 내에서는 Val Loss도 비슷하게 낮아 심각한 과적합은 아직 발생하지 않았다. Epochs를 더 늘리면 Val Loss가 다시 증가하는 전형적인 과적합 패턴이 나타날 것으로 예상된다.
- Dropout 적용 여부가 일반화 성능에 직접적인 영향을 미침을 확인했다.

## Lab 4 - 진자 주기 예측 + RK4 시뮬레이션

![결과](1-1.%20output/04_pendulum_prediction.png)

### 실험 목적

진자의 길이(L)와 초기 각도(θ₀)를 입력받아 주기 T를 예측하고, RK4 수치적분으로 진자 운동을 시뮬레이션한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 입력 | L (0.3~3.0m), θ₀ (5°~80°) |
| 출력 | 주기 T (s) |
| 물리 공식 | T = 2π√(L/g) + 큰 각도 타원적분 보정 |
| RK4 조건 | L=1m, θ₀=30°, dt=0.001s |

### 결과 및 분석

- **좌상단 (주기 예측)**: L=0.5m, 1.0m, 2.0m 각각에 대해 NN 예측(점)이 실제 값(실선)과 매우 잘 일치한다. 각도가 증가할수록 주기가 길어지는 비선형 관계도 잘 포착했다.
- **우상단 (학습 곡선)**: Train Loss와 Val Loss 모두 안정적으로 수렴한다.
- **좌하단 (RK4 시뮬레이션)**: L=1m, θ₀=30° 조건에서 각도가 시간에 따라 규칙적인 진동을 보인다. 감쇠 없는 이상적인 진자 운동이 정확히 재현됐다.
- **우하단 (위상 공간)**: 각도(θ)와 각속도(ω)의 관계가 닫힌 타원형 궤적을 형성한다. 에너지가 보존되는 이상적인 진자의 위상 공간 특성을 잘 나타낸다.

## 심화 과제 - 이론적 예측

> **미구현 사유**: Claude Pro 플랜의 컨텍스트 윈도우 한도 초과로 인해 추가 구현을 진행하지 못하였다. 기본 과제 GUI 앱 개발 과정에서 Superpowers Subagent-Driven Development를 통해 다수의 에이전트를 병렬 실행하며 대량의 토큰을 소모하였고, 심화 과제 구현 단계에서 세션 한도에 도달하였다. 이에 각 심화 과제의 이론적 예측 결과를 대신 기술한다.
>
> 단, 심화 기능(공기저항, L1/L2 정규화, 감쇠 진자, 추가 함수)의 **코드 구현은 완료**되어 있으며, GUI에서 파라미터를 조정하여 직접 실행 가능하다.

### Lab 1 심화: tanh(x), x³ 함수 추가

- **tanh(x)**: -1~1 사이 S자형 곡선으로, 활성화 함수와 동일한 형태. [128, 128] 아키텍처에서 MSE < 0.0001 수준으로 매우 정확한 근사가 예상됨. tanh 활성화 함수를 쓰는 네트워크가 같은 형태의 함수를 근사하므로 수렴이 빠를 것으로 예측.
- **x³**: 단조증가 함수로 범위가 넓어 정규화 필수. [-2π, 2π] 범위에서 값이 최대 ~248에 달하므로 데이터 스케일링 없이는 학습이 불안정할 것으로 예상. 정규화 후에는 [64, 64] 구조로도 MSE < 0.001 달성 가능할 것으로 예측.

### Lab 2 심화: 공기저항 추가 모델

- **운동 방정식**: m·dv/dt = -mg - k·v²
- 공기저항 계수 k가 클수록 최대 사거리 감소, 최적 발사각이 45° 미만으로 이동
- NN은 (v₀, θ, t, k) 4개 입력으로 (x, y) 출력하도록 확장 필요
- 공기저항 없는 경우보다 학습 난이도 상승 예상 (비선형성 증가)
- 동일한 [128, 64, 32] 구조 기준, 최소 500 epoch 이상 필요할 것으로 예측

### Lab 3 심화: L1/L2 Regularization 구현

- **L1 (Lasso)**: 일부 가중치를 정확히 0으로 만들어 희소한 모델 생성. Overfit 모델에 적용 시 Val loss가 Good Fit 수준으로 개선될 것으로 예측
- **L2 (Ridge)**: 가중치 크기를 전체적으로 줄여 부드러운 예측 곡선 생성. λ=0.01 기준으로 Overfit 모델의 Val/Train loss 격차가 50% 이상 감소할 것으로 예측
- Keras에서 `kernel_regularizer=keras.regularizers.l2(0.01)` 한 줄 추가로 구현 가능

### Lab 4 심화: 감쇠 진자 (Damped Pendulum)

- **감쇠 운동 방정식**: d²θ/dt² = -(g/L)sin(θ) - b·(dθ/dt)
- 감쇠 계수 b가 클수록 진동이 빨리 소멸, 위상 공간에서 타원이 안쪽으로 수렴하는 나선형으로 변화
- NN 입력: (L, θ₀, b), 출력: 유효 주기 T (진폭이 초기의 1/e이 되는 시점까지)
- 비감쇠 진자보다 데이터 다양성이 높아 학습 샘플 수를 3000개 이상으로 늘려야 할 것으로 예측
- MAPE 목표: b < 0.5 범위에서 5% 미만 달성 가능할 것으로 예측

## Lab 5 - 스프링-질량 고유 진동수 예측

![결과](1-1.%20output/05_spring_mass_frequency.png)

### 실험 목적

스프링 상수 k와 질량 m을 입력받아 고유 진동수 f = (1/2π)√(k/m) 를 신경망으로 학습하고, RK4 수치적분으로 실제 진동 운동을 검증한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 입력 | k (1~100 N/m), m (0.1~10 kg) |
| 출력 | f (Hz) |
| 네트워크 | [64, 32, 16, 1] ReLU |
| 학습 샘플 | 2000개 |
| Epochs | 1000 |
| RK4 조건 | k=10 N/m, m=1 kg, x₀=1 m |

### 결과 및 분석

- **좌상단 (NN 예측 vs 실제 f)**: 산점도에서 예측값이 이상적 예측 직선(빨간 점선) 위에 거의 완벽하게 분포한다. 전체 진동수 범위(0~3.2 Hz)에 걸쳐 오차가 매우 작다.
- **우상단 (학습 곡선)**: Train Loss와 Val Loss 모두 약 10¹에서 시작해 10⁻³ 수준까지 수렴했다. Val Loss가 Train Loss보다 약간 높고 노이즈가 있지만, 전반적으로 과적합 없이 안정적으로 수렴했다.
- **좌하단 (k 변화에 따른 f)**: m=1 kg 고정 시 k가 증가할수록 f가 증가하는 비선형(제곱근) 관계를 NN이 정확히 학습했다. 실제 곡선(파란 실선)과 NN 예측(빨간 점선)이 거의 완전히 겹친다.
- **우하단 (RK4 시뮬레이션)**: k=10, m=1 조건에서 10초 동안 약 3주기의 안정적인 사인 곡선 진동이 재현됐다. 에너지 손실 없는 이상적인 단순 조화 운동을 RK4가 정확히 적분함을 확인했다.
- **결론**: f = (1/2π)√(k/m) 공식은 두 변수의 곱/비율 관계로 비선형성이 있지만, [64, 32, 16] 규모의 네트워크가 충분히 학습 가능함을 확인했다.

## Lab 6 - 2D Ising Model 상전이 예측

![결과](1-1.%20output/06_ising_phase_transition.png)

### 실험 목적

Metropolis 알고리즘으로 생성한 2D Ising 모델의 온도-자화율 데이터를 신경망으로 학습하고, 상전이 임계 온도(Tc ≈ 2.269) 근방의 급격한 변화를 예측할 수 있는지 확인한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 격자 크기 | 10×10 |
| 온도 범위 | T = 0.5 ~ 5.0 (80개 구간) |
| Metropolis steps | 3000 per sample |
| 네트워크 | [32, 16, 1] tanh |
| Epochs | 500 |
| 임계 온도 | Tc ≈ 2.269 (J=1, k_B=1) |

### 결과 및 분석

- **좌상단 (|M| vs 온도 T)**: Monte Carlo 데이터(산점도)는 저온(T<2)에서 |M|≈1의 강자성 상태를 유지하다가, Tc≈2.27 부근에서 급격히 0으로 감소하는 2차 상전이를 명확히 보여준다. NN 예측(빨간 실선)이 이 S자 형태의 전이 곡선을 잘 따라간다.
- **우상단 (학습 곡선)**: Train Loss가 약 6×10⁻² 에서 2×10⁻² 수준으로 감소했다. 데이터 수가 80개로 적어 validation split 없이 전체 데이터로 학습했으며, 수렴은 빠르게 이루어졌다.
- **하단 격자 스냅샷**:
  - **저온 T=1.0 (강자성)**: 스핀이 대부분 같은 방향(파란색)으로 정렬된 질서 있는 상태.
  - **임계 T=2.27 (상전이)**: 두 방향 스핀이 섞이며 도메인 경계가 형성되는 임계 상태.
  - **고온 T=4.0 (상자성)**: 스핀이 무작위로 배열된 완전한 무질서 상태.
- **결론**: 신경망이 Metropolis 시뮬레이션 데이터만으로 상전이 임계 거동을 근사적으로 학습할 수 있음을 확인했다. 단, 10×10 소규모 격자의 유한 크기 효과로 인해 전이가 이론값보다 약간 완만하게 나타났다.

## Lab 7 - 양자 조화 진동자 에너지 준위 예측

![결과](1-1.%20output/07_quantum_oscillator_energy.png)

### 실험 목적

양자 조화 진동자의 에너지 공식 E_n = ℏω(n + 1/2) 를 신경망으로 학습하고 (ω, n) 입력으로 에너지 준위를 예측한다. 또한 Hermite 다항식을 이용한 파동함수를 시각화한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 입력 | ω (0.5~5.0), n (0~9 정수) |
| 출력 | E_n (에너지) |
| 단위 | ℏ = 1 |
| 네트워크 | [32, 16, 1] ReLU |
| 학습 샘플 | 3000개 |
| Epochs | 1000 |

### 결과 및 분석

- **좌상단 (에너지 준위 다이어그램, ω=1)**: n=0~9에 걸쳐 E_n = 0.5, 1.5, 2.5, ..., 9.5 의 균등 간격 에너지 준위가 수평선으로 표시된다. 양자화된 에너지 준위의 등간격 특성(ΔE = ℏω = 1)을 명확히 확인할 수 있다.
- **우상단 (파동함수 ψ_n(x))**: n=0~4에 대한 파동함수를 에너지 준위 위치에 오프셋하여 표시했다. n이 증가할수록 마디(node) 수가 n개로 증가하고, x=0 대칭/반대칭 특성이 교대로 나타나는 양자역학적 특성이 잘 재현됐다.
- **좌하단 (NN 예측 vs 실제 E_n)**: 예측값이 이상적 직선 위에 매우 정밀하게 분포한다. E_n = ω(n+0.5) 는 선형 관계이므로 작은 네트워크로도 완벽에 가깝게 학습된다.
- **우하단 (학습 곡선)**: Train Loss와 Val Loss가 10⁻² 에서 10⁻⁴~10⁻⁵ 수준까지 안정적으로 수렴했다. Val Loss가 간헐적으로 Train Loss보다 낮게 나타나는 것은 정규화 효과 또는 샘플링 변동으로 해석할 수 있다.
- **결론**: E_n = ω(n+0.5) 의 선형 관계를 NN이 거의 완벽히 학습했다. 이산적인 양자수 n을 연속형 실수로 취급해도 모델이 올바른 예측을 수행했다.

## Lab 8 - Kepler 궤도 예측

### 실험 목적

케플러 궤도 방정식 r = a(1-e²)/(1+e·cosθ) 를 신경망으로 학습하고, 장반경(a)과 이심률(e), 진근점 이각(θ)으로부터 직교좌표 (x, y)를 예측한다.

### 실험 설정

| 항목 | 값 |
|------|-----|
| 입력 | a (1~5 AU), e (0~0.9), θ (0~2π) |
| 출력 | x, y (AU) |
| 네트워크 | [128, 64, 32, 2] ReLU |
| 학습 샘플 | 3000개 |
| Epochs | 1000 |
| 테스트 조건 | a=2 AU, e=0.0 / 0.5 / 0.85 |

### 결과 및 분석

- **좌측 (궤도 예측)**: 3가지 이심률에 대해 실제 궤도(실선)와 NN 예측(점선)을 비교했다.
  - **원형 e=0.0 (파란색)**: 실제와 예측이 거의 완벽하게 일치한다. 원형 대칭성을 잘 학습했다.
  - **타원 e=0.5 (빨간색)**: 전반적인 타원 형태를 잘 따르며, 근일점~원일점 비율도 잘 재현됐다.
  - **긴 타원 e=0.85 (초록색)**: 이심률이 크면 근일점 근방에서 r이 급격히 변하는 비선형성이 강해진다. 전체 형태는 예측하지만 근일점 부근에서 약간의 편차가 발생한다. 이는 해당 영역의 훈련 데이터 밀도가 상대적으로 낮기 때문이다.
  - 태양(노란 점, 원점)이 타원의 초점에 올바르게 위치하는 것도 확인할 수 있다.
- **우측 (학습 곡선)**: Train Loss와 Val Loss가 10⁻¹에서 10⁻³ 수준으로 수렴했다. 노이즈가 있지만 지속적으로 감소하는 경향을 보인다. 출력이 2차원(x, y)이라 단일 출력보다 학습이 복잡하다.
- **결론**: 극좌표→직교좌표 변환 관계를 NN이 데이터로부터 학습할 수 있음을 확인했다. 이심률이 높을수록 비선형성이 강해져 예측 오차가 증가하며, 더 많은 데이터와 에포크로 개선 가능하다.
