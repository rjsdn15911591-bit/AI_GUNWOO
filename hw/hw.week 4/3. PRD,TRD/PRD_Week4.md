## 1. 제품 개요

### 1.1 목적

TensorFlow/Keras 신경망을 활용하여 물리 데이터를 학습하고 예측하는 PySide6 기반 GUI 애플리케이션을 개발한다. 사용자는 물리 파라미터를 GUI에서 직접 조정하고, 학습 결과를 실시간으로 시각화하며, 결과를 PNG로 저장할 수 있다.

### 1.2 배경

AI와 머신러닝 수업 Week 4 과제. 요구 조건:
- PRD + TRD 문서 작성
- Superpowers 스킬 활용 (brainstorming → writing-plans → executing-plans)
- PySide6 기반 GUI 구현
- TensorFlow/Keras 신경망 활용

### 1.3 개발 방법론

Claude Code **Superpowers** 스킬을 통해 설계-계획-구현 워크플로우를 적용하였다.

| 단계 | 스킬 | 산출물 |
|------|------|--------|
| 설계 | `superpowers:brainstorming` | 아키텍처, 물리 모델 결정 |
| 계획 | `superpowers:writing-plans` | 구현 순서, 검증 체크리스트 |
| 구현 | `superpowers:executing-plans` | 전체 소스코드 |
| 검토 | `superpowers:code-reviewer` | 품질 검증 |

## 5. 심화 과제 - 이론적 예측

> **미구현 사유**: Claude Pro 플랜의 컨텍스트 윈도우 한도 초과로 인해 심화 과제의 실험 결과를 별도로 생성하지 못하였다. 기본 과제 GUI 앱 개발 과정에서 Superpowers Subagent-Driven Development를 통해 다수의 에이전트를 병렬 실행하며 대량의 토큰을 소모하였고, 심화 과제 구현 단계에서 세션 한도에 도달하였다. 심화 기능의 **코드 구현은 완료**되어 있으며 GUI에서 직접 실행 가능하다.

### Lab 1 심화 예측: tanh(x), x³ 함수 추가

- **tanh(x)**: [128, 128] 아키텍처에서 MSE < 0.0001 수준 예상. tanh 활성화 함수와 동일한 형태의 함수를 근사하므로 수렴이 빠를 것으로 예측
- **x³**: [-2π, 2π] 범위에서 값이 최대 ~248으로 정규화 필수. 정규화 후 [64, 64] 구조로도 MSE < 0.001 달성 가능할 것으로 예측

### Lab 2 심화 예측: 공기저항 추가

- 운동 방정식: m·dv/dt = -mg - k·v²
- k가 클수록 최대 사거리 감소, 최적 발사각 45° 미만으로 이동
- [128, 64, 32] 구조 기준 최소 500 epoch 이상 필요 예측

### Lab 3 심화 예측: L1/L2 Regularization

- **L1**: 가중치 희소화, Overfit 모델 Val loss가 Good Fit 수준으로 개선 예측
- **L2 (λ=0.01)**: Overfit 모델 Val/Train loss 격차 50% 이상 감소 예측

### Lab 4 심화 예측: 감쇠 진자

- 감쇠 방정식: d²θ/dt² = -(g/L)sin(θ) - b·(dθ/dt)
- b가 클수록 위상 공간에서 타원 → 수렴 나선형으로 변화
- 학습 샘플 3000개 이상, b < 0.5 범위에서 MAPE 5% 미만 달성 가능 예측
