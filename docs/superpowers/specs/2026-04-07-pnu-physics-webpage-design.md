# 부산대학교 물리학과 웹페이지 디자인 현대화

**날짜:** 2026-04-07  
**대상:** `physics_web/` 디렉토리 (FastAPI 기반 기존 구현)  
**범위:** CSS/JS/HTML 전면 교체 — 백엔드 및 데이터 변경 없음

---

## 목표

기존 2000년대 스타일의 3컬럼 고정폭 레이아웃을 미니멀/클린 모던 디자인으로 전면 교체한다. MIT·Stanford 학과 사이트에서 영감을 받은 단일 컬럼 반응형 레이아웃으로 변경한다. Python/FastAPI 백엔드, Jinja2 템플릿, API 엔드포인트는 모두 그대로 유지한다.

---

## 레이아웃 구조

단일 컬럼 흐름, 최대 너비 1200px, 반응형:

1. **HEADER** (sticky) — 로고 + 수평 GNB 네비게이션 + 드롭다운
2. **MARQUEE BAR** — CSS 애니메이션 기반 공지 스크롤
3. **HERO SECTION** — 학과 소개 문구, 옅은 배경
4. **STATS BAR** — 교수/학생/연구실/논문 수치 카드 4개 (카운트업 애니메이션)
5. **NEWS + EVENTS** — 2컬럼 그리드 (공지사항 탭 | 학과 행사 목록)
6. **FACULTY** — 전공별 탭 필터 + 카드 그리드 (4열)
7. **FOOTER** — 주소, 연락처, 링크

기존 좌/우 사이드바 제거. bkit 텍스트 분석기 위젯 제거.

## 컴포넌트 스타일

- **카드:** `border-radius: 12px`, 미세 섀도우, hover 시 translateY(-4px) 전환
- **탭:** 하단 라인 방식 (언더라인 액티브 표시)
- **교수 카드:** 이름 + 직위 배지 + 전공 + 이메일, 클릭 시 상세 토글 유지
- **직위 배지:** 교수(파랑) / 부교수(보라) / 조교수(초록) 색상 구분 유지
- **버튼:** `border-radius: 8px`, 포인트 컬러

## 인터랙션 & JS

### 유지
- 탭 전환 (공지사항/학사공지/취업정보, 교수진 전공별 필터)
- 교수 행/카드 클릭 → 상세 정보 토글
- 실시간 시계 (헤더 또는 제거 고려)
- 검색 (`/api/search` API 호출)
- 방문자 카운터

### 개선
- 스크롤 시 헤더 배경 투명 → 흰색 전환
- 통계 카운터 카운트업 애니메이션 (IntersectionObserver 기반)
- 탭 전환 시 fade 효과
- 교수 카드 hover 애니메이션

### 제거
- bkit 텍스트 분석기 위젯 (학과 사이트와 무관)
- 기존 `<marquee>` 태그 → CSS animation으로 대체
