# PRD — AI 음식 분류기 (Product Requirements Document)

**버전:** 1.0  
**작성일:** 2026-04-15  
**서비스 URL:** https://ai-gunwoo.vercel.app

---

## 1. 제품 개요

### 배경
스마트폰으로 음식 사진을 찍으면 어떤 음식인지 즉시 알고 싶은 수요가 있다. 기존 서비스는 서버로 이미지를 전송하기 때문에 느리고 개인정보 우려가 있다. 본 제품은 브라우저 내 딥러닝 추론으로 이미지를 외부 서버에 전송하지 않고 실시간으로 음식을 분류한다.

### 제품 목표
- 별도 앱 설치 없이 웹 브라우저에서 즉시 사용
- 카메라 촬영 또는 파일 업로드로 간편하게 음식 분류
- 분류 결과를 한국어로 제공하여 접근성 향상
- 프리미엄 구독으로 고급 AI 분석 기능 제공

## 2. 사용자 페르소나

| 구분 | 설명 |
|------|------|
| 일반 사용자 | 식단 관리, 음식 탐구에 관심 있는 일반인. 하루 5회 무료 분류로 충분 |
| 프리미엄 사용자 | 식품 관련 종사자, 다이어터, AI 기능에 관심 있는 사용자. 무제한 분류 + 고급 분석 필요 |

## 3. 핵심 기능 요구사항

### 3.1 음식 분류 (모든 사용자)
- 이미지 업로드 (JPG, PNG, WEBP) 또는 카메라 촬영으로 입력
- 버튼 클릭 시 분류 시작 (자동 분류 없음)
- Top-5 분류 결과를 신뢰도(%) + 한국어 레이블로 표시
- 비음식 이미지 감지 시 경고 메시지 표시
- 음식 항목만 결과에 표시 (비음식 클래스 필터링)

### 3.2 사용량 관리
- 무료 사용자: 일 5회 분류 제한
- 한도 초과 시 업그레이드 유도 메시지 표시
- 사용 잔여 횟수 실시간 표시

### 3.3 인증
- 이메일 회원가입 / 로그인
- Google OAuth 소셜 로그인
- 로그인 상태, 계정 이메일, 플랜 등급 헤더 표시
- 로그아웃 기능

### 3.4 프리미엄 기능
- 무제한 분류
- TTA 5-crop 앙상블 (정확도 향상)
- Grad-CAM 히트맵 (모델 주목 영역 시각화)
- 불확실도 배지 (표준편차 σ로 예측 안정성 표시)
- 분류 히스토리 저장 및 조회

### 3.5 결제
- Polar.sh 구독 결제 (₩7,000/월)
- 결제 완료 시 플랜 자동 업그레이드
- 구독 취소 시 플랜 자동 다운그레이드

## 4. 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 추론 속도 | WebGL 백엔드 기준 3초 이내 |
| 프라이버시 | 이미지 외부 서버 전송 없음 (브라우저 내 추론) |
| 호환성 | Chrome, Safari, Firefox 최신 버전 |
| 반응형 | 모바일, 태블릿, 데스크톱 모두 지원 |

## 5. 사용자 흐름

```
랜딩 페이지
    ↓ 로그인/회원가입
앱 페이지 (/app)
    ↓ 이미지 업로드 또는 카메라 촬영
    ↓ [분류 시작] 버튼 클릭
분류 결과 표시
    ├── Top-5 음식 후보 + 신뢰도
    ├── (프리미엄) Grad-CAM 히트맵
    └── (프리미엄) 불확실도 배지
```

---

## 6. 성공 지표

- 분류 정확도: 일반 음식 이미지에서 Top-1 정확도 60% 이상
- 무료→프리미엄 전환율: 5% 이상
- 월간 활성 사용자(MAU) 성장
